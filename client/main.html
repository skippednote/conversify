Conversify

About

Conversify is a web-based real-time chat application, designed for chatting in public. There is a pleasing design for both desktop and mobile browsers. Conversify supports Emoji too.

Conversify is built on top of Open Source technologies. We are using Meteor.js on both client and server side, apart from that we are using Bootstrap for some styling and Moment.js for generating beautiful timestamps. For authentication we are using Twitter OAuth which is the source for user meta information like Avatar and Name.

A project created, maintained, and designed by Bassam Ismail.
Many thanks to Alvi Murtaza and other contributors.
