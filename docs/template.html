    We will start with a base application template that will get rendered throughout the application. Meteor will be adding the doctype and the html tags automatically and here in the application template we will be skipping those. As for the head tags, these will be retrived from various templates and concatenated into one.

    <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title></title>
</head>
<body>

              </a>

                    {{loginButtons}}

        </div>
